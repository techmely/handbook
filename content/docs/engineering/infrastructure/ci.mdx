## Overview CI System

### Frontend CI

1. Linting: 
  - Use `biomejs` to format code and check code follow the team's standard
  - Use [`bundlesize`](https://github.com/siddharthkp/bundlesize) to check the size of the bundle 
2. Typecheck
3. Unit Test
  - Use vitest to run unit test
4. e2e test
  - Use playwright to write e2e test
5. CRUX - Lighthouse Page Speed Api
  - Use crux api to check the page speed in the real world
